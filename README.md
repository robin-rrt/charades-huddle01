# charades-huddle01
a simple charades playing app, where users stake 20 DAIx, and streams the funds to opposition throughout the time they are guessing.

## How to run
1. `git clone [http link]`
2. `cd charades-huddle01`
3. `npm install`
4. `npm run dev`
5. view on `localhost:3000`
